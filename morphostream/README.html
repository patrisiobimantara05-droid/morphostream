# MorphoStream QGIS Plugin
## Deskripsi
MorphoStream adalah plugin QGIS untuk analisis morfometri sungai dan DAS.
Plugin ini menyediakan algoritma modular untuk menghitung parameter
hidrologi penting secara otomatis.
### Fitur Utama
- **Sinuosity Index (SI)** menghitung kelengkungan sungai utama.
- **Drainage Density (Dd) & Bifurcation Ratio (Rb)** analisis jaringan
sungai.
- **Parameter DAS (Ff, Rc, Re, Cc)** analisis bentuk dan kompaksi DAS.
